## 😁翻译原则😊

- 采用机翻+润色的方式进行，因为我是个日语渣，翻译初稿来自百度为主+彩云小译为辅的结果

- 主要采用意译的方式，因为本来就是电波向作品，很多地方并不好直译

- 不会翻的地方使用 // 标出，希望老哥们帮帮忙（等上传sence01就能看到了，现在还在做…）
